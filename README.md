So yeah, it's a pomodoro timer.
Built with React.

This is solution for https://roadmap.sh/projects/pomodoro-timer .

To run this project locally:

```bash
git clone https://github.com/cxnexion/P-Time.git
cd P-Time
npm install
npm start
```
